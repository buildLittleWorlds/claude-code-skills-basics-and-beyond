Curriculum
Build once, benefit every time. A hands-on curriculum for building skills, hooks, agents, and workflows in Claude Code.

Curriculum Overview

This site is a reading companion for the Claude Code Skills curriculum — twenty-six lessons across three phases that teach you to build skills, hooks, custom subagents, agent teams, and a complete task-management cockpit. Each page transforms a lesson.md source file into a browsable reference with syntax-highlighted code blocks, step-by-step walkthroughs, and copy-ready terminal commands. Follow the sequence from Course 1 through the Daily Rhythm, or jump directly to the topic you need.

How It Works

Read & Build

Each course page pairs concept explanations with a hands-on project. You read the page, then build the skill (or hook, or agent) yourself in Claude Code.

Copy-Ready Code

Terminal commands, YAML frontmatter, and full SKILL.md files are syntax-highlighted and copyable. Hover over any code block to reveal the copy button.

Progressive Difficulty

Courses progress from beginner (a single SKILL.md file) through intermediate (hooks, testing, workflows) to advanced (subagents, agent teams, a full plugin).

Three Phases

Phase 1 covers skills fundamentals in a single terminal. Phase 2 adds tmux, subagents, and multi-agent orchestration. Phase 3 builds a complete task-management cockpit with six skills, two hooks, and a one-command launch.

Phase 1: Skills Fundamentals

Seven courses that take you from your first SKILL.md file to hook-powered automation — all within a single Claude Code session. No tmux required.

1. Your First Skill

Beginner · 15 min

Build a daily-standup skill. Learn what skills are, where they live, YAML frontmatter, and naming rules.

2. Descriptions & Triggers

Beginner · 25 min

Build a code-review-checklist skill. Master the description formula, trigger phrases, and progressive disclosure.

3. Full Skill Anatomy

Beginner-Intermediate · 35 min

Build a meeting-to-actions skill with scripts, references, and assets. Learn the complete folder structure.

4. Testing & Iteration

Intermediate · 40 min

Build a pr-description skill with a structured test plan. Learn triggering tests, functional tests, and iteration signals.

5. Advanced Features

Intermediate · 45 min

Build an investigate-issue skill. Learn $ARGUMENTS, dynamic context injection, and subagent execution.

6. Multi-Step Workflows

Intermediate · 45 min

Build a project-scaffolder skill. Learn sequential workflows, validation gates, and error handling patterns.

7. Hooks

Intermediate · 35 min

Build hook scripts and a safe-deploy skill. Learn deterministic control: PreToolUse, PostToolUse, command and prompt hooks.

Bridge: Terminal Workspace

7½. Your Terminal Workspace

Intermediate · 20 min

A prerequisite checkpoint for the advanced courses. Verifies you can split panes, manage detached sessions, and use send-keys and capture-pane in tmux.

Phase 2: Integrated Advanced

Four courses that merge skills with tmux orchestration — subagents running in separate contexts, CLI tool wrappers, agent teams working in parallel, and a capstone plugin that ties everything together.

8. Custom Subagents

Advanced · 50 min

Build a test-runner and security-reviewer agent. Learn subagent config, tool restrictions, model selection, and tmux monitoring.

9. CLI Integration

Advanced · 50 min

Build a github-release-notes skill and a dx-review wrapper script. Learn to combine skills with gh, git, and shell scripting.

10. Agent Teams

Advanced · 60 min

Orchestrate parallel agents with shared task lists, mailbox messaging, and a tmux monitoring dashboard. Build a multi-reviewer code review team.

11. The Capstone

Advanced · 75 min

Build a complete engineering-workflow plugin: four composable skills, two agents, hooks, and a tmux session script that orchestrates the full pipeline.

Phase 3: Task Workflow Mastery

Fifteen lessons that build a complete task-management cockpit from scratch — six skills, two hooks, a startup script, and a tmux layout that launches with one command. Every pattern from Phases 1 and 2 comes together here.

1. Talking to Panes

Intermediate · 20 min

Use tmux send-keys -t addressing to control panes you are not focused on. Build a 3-window task layout.

2. Capture Pane

Intermediate · 20 min

Read output from non-focused panes with capture-pane. Build monitoring one-liners that watch Claude’s activity.

3. Jumping Between Worlds

Intermediate · 20 min

Manage dual sessions (tasks + coding) with rapid switching. Practice the rhythm the cockpit automates later.

4. /next Skill

Intermediate · 25 min

Build your first task skill. Reads pending tasks, sorts by due date, and displays a formatted queue.

5. /done Skill

Intermediate · 30 min

Complete tasks with $ARGUMENTS: move to archive, append log entry, show a floating display-popup confirmation.

6. /triage Skill

Intermediate · 30 min

Process inbox items with dynamic context and reference-file routing rules. Classify, tag, and route to pending.

7. Startup Script

Intermediate · 25 min

Write a ~/bin/tasks script that launches the cockpit — three windows, reattach guard, and auto-briefing.

8. /today Skill

Intermediate-Advanced · 25 min

Morning briefing in a forked Explore agent: overdue items, due today, coming this week, and suggested focus.

9. /overdue Skill

Intermediate-Advanced · 25 min

Four-tier escalation rules from gentle reminder to blocking popup. Teach Claude to nag proportionally.

10. Compound Reminder

Intermediate-Advanced · 25 min

A Stop hook that checks whether MEMORY.md was updated after completing a task. Includes the infinite-loop guard.

11. File Protection

Intermediate-Advanced · 20 min

A PreToolUse hook that blocks edits to log and archive files, directing users to /done instead.

12. Cockpit Layout

Advanced · 30 min

Assemble the full 4-pane HQ: Claude, shell, inbox watcher, and pending/overdue dashboard with a custom status bar.

13. Side-Gigs Domain

Advanced · 20 min

Integration test: add freelance and personal tasks, then verify the entire pipeline generalizes beyond seed data.

14. Session Summary

Advanced · 20 min

Build /session-summary to close the daily loop: tasks completed, decisions made, and focus for tomorrow.

15. Daily Rhythm

Advanced · 30 min

The complete integration test. Every skill, hook, and tmux pattern verified end-to-end with a test script.

Getting Started

All you need is Claude Code installed and a terminal. Start with Course 1: Your First Skill — you'll have a working skill in under fifteen minutes.

Source files: The lesson source files and all runnable skill code live in the GitHub repository. Each course directory under courses/ contains a lesson.md and a skill/ folder you can install directly.
